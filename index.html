Features

Lua adds a fully functional economy and levelling system, with its' primary currency unit being the ‚Äòcoin‚Äô. Coins can be used to purchase tradable items. Note that coins have no real world value. Both coins and levelling experience are obtainable through sending messages that are visible to the bot.

Lua's levelling system includes a unique prestige mechanic, where users can reset their level to 0 after reaching level 50 in order to gain a prestige rank. This gives server admins fine-tuned control over levelling progression and reward, which very few alternatives provide. More informat ion can be found in the Levelling section.

To ensure that Lua can coexist with other bots, Lua's prefix can be edited on a server by server basis.

Levelling

Levels and prestiges are stored individually by server.

Each user will only have 1 message count to their message total every minute

Users may prestige only if their level is greater than or equal to 50

When prestiging, any messages after Level 50 will not be transferred to the next prestige

Note that users may choose to not prestige and continue beyond level 50

It takes approximately 2500 minutes of messaging to reach the 1st prestige

Levelling Functions

Converting message total to level:

\[f\left(x\right)=\sqrt{x-5}\cdot0.9^{r}\]

x is the message total

p is the number of prestiges

Calculating for # of messages required to reach desired level:

\[g\left(x,\ p\right)=\frac{x^{2}}{0.9^{2p}}+5\]

x is the desired number of levels

p is the number of prestiges

List of supported commands:

Lua's prefix defaults to !.

lua prefix: gets the server's prefix. this command doesn't require a prefix

Key

{arg}: argument

*{arg}: optional argument

Utility Commands

!ping: gets ping

!help: helpful information

!commands: gets the list of commands

!whois: gets user_id

!bin {number}: converts a base-10 number to binary

!oct {number}: converts a base-10 number to octal

!hex {number}: converts a base-10 number fo hexadecimal

!reverse {message}: reverses a message

!launches: gets the current # of launches

Economy Commands

!claim: claims 50 coins

!balance *{user}: gets the user's balance

!coinflip {wager}: performs a coinflip with wager

!gamble {wager}: multiplies the wager by (-1:1) and adds to user's balance

!pay {user} {amount}: pays user some amount of coins

!shop: lists available shop items

!info {itemid}: gets an item's description

!buy {itemid} *{amount}: purchases an item

!inventory *{userid}: view someone's inventory

!use {itemid}: uses an item

!jobs: lists all the available jobs

!work *{jobid}: change jobs or work

!gift {itemid} {userid} *{amount}: gifts item(s) to a user

!recipes: lists all available recipes

!craft {itemid} *{amount}: crafts the specified amount of items

Levelling Commands

!level *{user_id}: gets the relevant user's level

!prestige: gain 1 prestige rank and resets the user's level

Administrator Commands

These commands require admin permissions

!prefixset {new_prefix}: sets the prefix in a particular server to new_prefix

!disable {channel}: disables Lua in the specified channel

!enable {channel}: enables Lua in the specified channel
